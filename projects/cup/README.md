# Cup 项目

这是一个杯子产品的营销海报项目，展示了316无缝内胆一体成型的杯子产品。

## 项目结构

```
projects/cup/
├── original/           # 原始设计文件
│   ├── cup.json       # Fabric.js设计数据
│   └── cup.svg        # SVG参考文件
├── resources/          # 下载的图片资源
│   ├── cup_bg_1.png   # 背景图片1
│   ├── cup_bg_2.png   # 背景图片2
│   └── cup_decor_*.png # 装饰图片
├── output/            # 渲染输出
│   └── cup_poster.png # 生成的海报
├── cup_protocol.json  # Skia渲染协议
├── download_cup_images.sh # 图片下载脚本
├── extract_positions.py   # 位置信息提取脚本
├── extract_text.py        # 文本信息提取脚本
└── README.md          # 项目说明
```

## 设计元素

### 图片元素
- **背景图片**: 2张背景图片，尺寸800x800和836x836
- **装饰图片**: 14张装饰图片，包括产品展示、标签等

### 文本元素
- **主标题**: "夜奶无忧"、"一杯五用" - 大号字体，深色
- **副标题**: "316无缝内胆一体成型" - 白色字体
- **功能说明**: "防胀气"、"抗摔打" - 不同颜色
- **促销信息**: "双十一提前购"、"送杯刷+吸管刷+奶嘴刷"
- **价格信息**: "¥59.8"、"预估到手价"
- **赠品信息**: "赠送运费险"、"八重大礼包"

## 技术特点

- **画布尺寸**: 800x800像素
- **字体**: SourceHanSansCN-Normal
- **颜色方案**: 白色、深灰色、绿色系
- **透明度**: 部分装饰图片使用透明度效果
- **缩放**: 部分文本使用缩放变换

## 使用方法

1. 下载图片资源：
   ```bash
   ./download_cup_images.sh
   ```

2. 渲染海报：
   ```bash
   ./build/skia
   ```

3. 查看结果：
   ```bash
   open projects/cup/output/cup_poster.png
   ```

## 设计亮点

- 突出产品核心卖点：316无缝内胆、防胀气、抗摔打
- 清晰的价格展示和促销信息
- 丰富的赠品信息增加购买吸引力
- 专业的视觉层次和色彩搭配
