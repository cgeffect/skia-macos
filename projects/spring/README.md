# Spring 春天海报项目

## 项目描述

这是一个关于春天的海报设计，主题为"用最甜的方式唤醒春天"，包含丰富的春天元素和装饰。

## 文件结构

```
projects/spring/
├── original/                    # 原始设计文件
│   ├── spring.json             # Fabric.js 设计文件
│   └── spring.svg              # SVG 参考文件
├── resources/                   # 图片资源
│   ├── spring_bg_1.png         # 背景图片
│   ├── spring_main.png         # 主图
│   ├── spring_decor_*.png      # 装饰图片
│   ├── spring_text_bg_*.png    # 文字背景
│   └── spring_element_*.png    # 装饰元素
├── output/                      # 输出文件
│   └── spring_poster.png       # 渲染结果
├── spring_protocol.json        # 渲染协议文件
├── download_spring_images.sh   # 图片下载脚本
└── README.md                   # 项目说明
```

## 设计元素

### 主要文本
- **标题**: "用最甜的方式唤醒春天"
- **日期**: "2022.04.02"

### 图片元素
- **背景**: 春天主题背景
- **主图**: 春天场景主图
- **装饰**: 多个春天元素装饰
- **文字背景**: 文字区域的背景装饰
- **边框元素**: 海报边框装饰

### 字体
- **站酷快乐体**: 用于标题和日期

## 使用方法

### 1. 下载图片资源
```bash
./download_spring_images.sh
```

### 2. 渲染海报
```bash
# 使用我们的渲染库
./build/skia
```

### 3. 查看结果
渲染完成后，海报将保存在 `output/spring_poster.png`

## 技术规格

- **画布尺寸**: 1242 x 1660 像素
- **背景色**: 白色 (#FFFFFF)
- **输出格式**: PNG
- **质量**: 100%

## 设计特点

1. **春天主题**: 以春天为主题，色彩明亮温暖
2. **层次丰富**: 包含背景、主图、装饰等多个层次
3. **文字突出**: 标题文字使用白色，在深色背景上突出显示
4. **装饰精美**: 包含多个精致的装饰元素
5. **边框设计**: 上下边框装饰，增加整体美感

渲染后的海报将呈现：
- 清晰的春天主题背景
- 突出的标题文字
- 丰富的装饰元素
- 完整的边框设计
- 高质量的图片输出
